## ✨ Features

✅ Chat with CSV using natural language  
✅ AI-powered data analysis  
✅ Auto-generated visualizations (Matplotlib + Seaborn)  
✅ Powered by Groq LLM (Llama 3.3 70B)  
✅ Built with Streamlit UI  
✅ Fast & interactive insights  

---

## 🧠 Tech Stack

- 🐍 Python  
- ⚡ Groq (Llama-3.3-70b)  
- 🔗 LangChain  
- 📊 Pandas  
- 📈 Matplotlib & Seaborn  
- 🌐 Streamlit

## ⚙️ Installation

```bash
git clone https://github.com/your-username/groq-csv-ai-agent.git
cd groq-csv-ai-agent
````

```bash
pip install -r requirements.txt
```

---

## 🔑 Setup

Get your API key from Groq and run:

```bash
streamlit run app.py
```

---

## 🖥️ Usage

1. Enter your **Groq API Key**
2. Upload your CSV file
3. Ask questions like:

   * "What is the average sales?"
   * "Show top 10 categories"
4. Generate visualizations instantly 📊
